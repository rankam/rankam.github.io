Projects

Simplex

  • Wanting to learn more about D3js and exploratory data analysis (EDA), Simplex was created as a platform that allows the user to quickly perform basic EDA. Simplex believes that ease of interpreting analysis results is an often overlooked feature in data analysis applications. Simplex aims to make interpreting results easy through providing intuitive explanations and using familiar color indicators in results. Simplex is under development in its alpha stage - check it out here.

Napoleon's Army Marches to Russia

  • The inspiration for recreating Charles Minard'sTableaux Graphiques et Cartes Figuratives struck while I read Edward Tufte's The Visual Display of Quantitative Information. Searching the internet, I was unable to find a recreation of the graphic using D3js and I though I would give it a shot. The graphic requires a variety of techniques I thought it would be an excellent way to improve my D3js skills as well as learn more about creating effective graphics.
