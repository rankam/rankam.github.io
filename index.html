Simplex

  • Wanting to learn more about D3js, ajax, and building data analysis web applications, Simplex was created as a platform that allows the user to quickly perform several basic analysis techniques. Simplex believes that ease of interpreting analysis results is an often overlooked feature in data analysis applications and aims to make interpreting results easy through providing intuitive explanations and using familiar color indicators in results. The project is under development in its alpha stage - check it out here.
